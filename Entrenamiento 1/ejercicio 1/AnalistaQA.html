Curso: Analista de Calidad de Software (QA)

Competencias Adquiridas

Este curso me ha capacitado para garantizar la estabilidad y correcto funcionamiento de aplicaciones financieras:

  • Diseño de Casos de Prueba (Test Cases): Capacidad para crear guías paso a paso que verifiquen el funcionamiento de una transacción o un alta de usuario.
  • Ciclo de Vida del Defecto (Bug Life Cycle): Habilidad para detectar, reportar y realizar el seguimiento de errores técnicos hasta su resolución.
  • Testing de Caja Negra: Competencia para probar la aplicación desde el punto de vista del usuario final, asegurando que la interfaz sea intuitiva y sin errores.
  • Pruebas de Regresión: Capacidad para verificar que las nuevas actualizaciones de la app no rompan funciones que ya andaban bien.
  • Uso de Jira y Trello: Manejo de herramientas de gestión de proyectos para documentar y priorizar fallos encontrados.
